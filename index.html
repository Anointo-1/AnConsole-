Reterio Console

Welcome to the official launch screen for the Reterio virtual console.

About the Console

This was made by anointo by the way :)

This is a custom-built, web-based virtual console created entirely from scratch. Designed by a solo indie developer, it simulates a retro gaming environment complete with its own logic and interface.

The system currently features an 8-button layout (D-Pad, A, B, Start, and Select). I am continuously working on the engine, and future updates will expand its capabilities.

Current & Upcoming Features:

  • Dual-player controller support (Version 1.1+)
  • Customizable console numbering
  • Touch screen integration (Upcoming)
  • Virtual input cord simulation (Upcoming)
  • Experimental motion controls (Upcoming)

If you encounter any bugs, especially on the PC version (as this was optimized heavily for mobile), please report them on My YouTube Channel.
